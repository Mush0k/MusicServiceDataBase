Курсовая работа по дисциплине «Проектирование баз данных» (5 семестр)

Описание проекта:
Проектирование и реализация базы данных для музыкального сервиса. Система поддерживает учет артистов, альбомов, треков и сложных связей (коллаборации Many-to-Many, многожанровость)

Состав репозитория:
KibbyModelPNG.png — Визуальная ER-диаграмма базы данных
KibbyModel.mwb — Исходный файл модели в MySQL Workbench
kibbyDump.sql — Полный SQL-дамп структуры и тестовых данных для быстрого импорта
ПОСЯНИТЕЛЬНАЯ ЗАПИСКА - полная информация о проектировани базы данных для музыкального серсива
